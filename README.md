# **🎥 NIMF (Not In My Frame)**

**실시간 스트리밍 콘텐츠 보호 AI 솔루션**

---

## **🔍 서비스 개요**

> **AI 비서형 실시간 감시 + 보호 + 피드백 시스템**

WebRTC 기반의 실시간 스트리밍 보조 솔루션으로, 스트리머가 인지하지 못한 리스크(욕설, 금지 물품, 타인 노출 등)를 AI가 실시간으로 감지하여 즉각적인 알림과 조치를 제공합니다.

---

## **🛠 주요 기능**

**📹 실시간 영상 모니터링**

- 스트리머 외 인물 자동 감지
- **위험/금지 물품** 탐지 및 **자동 블러 처리** (담배, 칼, 주류 등)

**🔊 실시간 음성 모니터링**

- **욕설·비속어 자동 인식**
- 해당 구간 **자동 음소거 처리**

**🔔 알림 및 피드백**

- **시각 알림**: 방송에만 보이는 미니 경고창
- **소리 알림**: 스트리머만 들을 수 있는 이어폰 사운드
- **행동 유도 멘트**: “흡연 노출 감지됨, 자리 조정 권장” 등

---

## **🧱 주요 기술 개요**
| **요소** | **적용 기술** |
| --- | --- |
| 실시간 스트리밍 | WebRTC (영상/음성 전송) |
| 객체 감지 | YOLOv8 (금지 물품, 인물 탐지) |
| 음성 분석 | STT + 욕설 필터링 모델 |
| UI 알림 | WebSocket 기반 경고창 & 사운드 알림 |
| 인페인팅 처리 | Segment Anything + Fast Blur (ONNX, FastAPI 등 활용) |
